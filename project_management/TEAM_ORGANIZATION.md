## Decision Process

Technical decisions were discussed before implementation whenever a feature could impact both contributors.

Examples:

* Choice of graphical library
* Build system organization
* Highscore implementation
* Deployment workflow

## Collaboration Method

* Dedicated branches for each contributor
* Regular merges into main branch
* Continuous communication through messages and in-person work sessions at 42
* Mutual code review before major integrations
