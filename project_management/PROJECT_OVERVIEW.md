# Project Overview

## Objective

The goal of this project was to recreate the classic Pac-Man arcade game in Python while respecting the project requirements:

* Object-oriented architecture
* Maze generation using the assigned A-Maze-ing package
* JSON configuration system
* Persistent highscore management
* Graphical interface
* Packaging and deployment on Itch.io
* Robust error handling

## Development Approach

The project was developed collaboratively using Git branches and frequent communication.

The team first focused on creating a stable technical foundation:

* Project structure
* Dependency management
* Makefile
* Configuration parser
* Highscore system
* Build and deployment pipeline

Once the foundation was established, gameplay and graphical features were implemented incrementally.

## Main Technical Choices

* Python 3.10+
* Pygame for graphical rendering
* UV for dependency management
* PyInstaller for packaging
* JSON-based configuration files
* JSON-based persistent highscores

## Collaboration

Development tasks were divided according to each member's priorities and progress. Communication was mainly done through direct discussions and regular synchronization messages.
